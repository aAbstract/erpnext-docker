# ERPNext Admin Manual

### Users Managment

#### Add User
1. Navigate to `/app/users`
2. Click on `+ Add User`

#### Change User Password
1. Navigate to `/app/users`
2. Select User from Shortcuts Panel
3. Search for the Target User
4. Go to `Settings > Change Password` Form
